## 🔑 获取完整功能

### 配置 Claude API 密钥

**1. 获取密钥**
- 访问: https://console.anthropic.com/
- 注册/登录
- 创建 API Key
- 复制密钥（格式: `sk-ant-api03-...`）

**2. 配置环境变量**
```bash
# 编辑 .env 文件
nano .env

# 找到这一行:
ANTHROPIC_API_KEY=sk-ant-api03-your-api-key-here

# 替换为真实密钥:
ANTHROPIC_API_KEY=sk-ant-api03-xxxxx（您的密钥）

# 保存并退出（Ctrl+O, Enter, Ctrl+X）
```

**3. 重启后端服务器**
```bash
# 在后端终端按 Ctrl+C 停止
# 然后重新启动
npm run server
```

## 🔍 故障排查

### 无法访问前端
1. 确认前端服务器在运行
2. 尝试: http://127.0.0.1:5173
3. 检查端口占用: `lsof -i :5173`

### 后端API错误
1. 查看错误日志: `cat logs/error.log`
2. 检查环境配置: `./配置助手.sh`
3. 重启后端服务器

### 翻译失败
1. 检查是否配置 API 密钥
2. 查看错误提示
3. 简化模式只能分析，不能翻译
