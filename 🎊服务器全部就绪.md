## 🔑 获取完整功能

### 配置 Claude API

1. **获取密钥**
   - 访问: https://console.anthropic.com/
   - 创建API密钥

2. **配置 .env**
   ```bash
   # 用编辑器打开
   nano .env
   
   # 修改这一行
   ANTHROPIC_API_KEY=sk-ant-api03-your-real-key-here
   ```

3. **重启后端**
   ```bash
   # 在后端终端按 Ctrl+C
   # 然后重新运行
   npm run server
   ```
